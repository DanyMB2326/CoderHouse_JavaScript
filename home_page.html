Sistema de Gestión "El Buen Sabor"

Descripción del Simulador

Este proyecto es una primera aproximación a un sistema de gestión para un restaurante. Permite realizar tareas administrativas básicas mediante una interfaz basada en cuadros de diálogo de JavaScript.

Actualmente, el sistema permite registrar nuevos empleados, tomar órdenes a los clientes (lo cual descuenta automáticamente el stock de los ingredientes/productos), y generar una factura final con cálculo de IVA. La interacción principal se realiza a través de la consola del navegador y ventanas emergentes (prompts y alerts).

Abre la consola del navegador (F12) para ver registros detallados.
